Détails de la vue orbitale

Pourquoi deux modes dans cette page ?

04aMode Orbital · Visuel

  • Métaphore complète : BTC et Gold comme soleils, sources en orbite
  • Tap sur un satellite = panneau de détail s'affiche en bas (ici "ETF BTC")
  • Couleurs par catégorie : vert flux, bleu macro, jaune inflation, rouge stress, violet positionnement
  • Idéal pour : prendre du recul, comprendre le système globalement, présenter à quelqu'un
  • Limite mobile : 8 satellites max visibles confortablement
  • Bouton "Liste" en haut à droite pour basculer vers le mode 04b

04bMode Liste · Influence

  • Trié par poids de la source dans le signal calculé
  • Toggle entité en haut : BTC / Gold / Silver / Tous
  • 3 niveaux d'influence : forte (vert), moyenne (orange), faible/inactive (gris)
  • Indique l'impact mesuré de chaque source : +0.18, +0.14, etc. → contribution à la confiance
  • Idéal pour : comprendre pourquoi un signal est ce qu'il est, identifier les sources dominantes
  • Sources désactivées visibles mais grisées (ex: COT GOLD par ADR-018 P2)

05Choix techniques mobile

  • Orbites en CSS pur (border-radius + border) ou SVG si besoin de courbes plus précises
  • Satellites positionnés en absolute avec coordonnées calculées
  • Sur mobile : pas d'animation de rotation orbitale (trop CPU) — version statique
  • Web/desktop éventuel : animation lente de rotation possible (CSS @keyframes)
  • Couleurs sémantiques = cohérentes avec les engines backend

06Place dans le parcours

  • Accès : depuis le bouton "Vue d'ensemble orbitale" dans Signals (vue 01)
  • Pas dans la nav principale : c'est une vue contextuelle, pas un onglet permanent
  • Usage occasionnel : quand tu veux prendre du recul, pas pour décider rapidement
  • Bouton retour ramène vers Signals
  • Vue marketing/démo : c'est cette page qu'on montre quand on veut illustrer Tik
