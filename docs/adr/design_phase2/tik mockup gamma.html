Détails design

Ce qui rend cette interface distinctement Tik

01Identité cosmique

  • Mini-soleils BTC/Gold dans chaque carte (orbes lumineuses)
  • Glow d'arrière-plan diffus dans chaque carte de signal
  • Étoiles subtiles sur tout le fond de l'écran
  • Fraunces italique pour les états (Risk-off, Expansion) → caractère poétique
  • Couleur ambre/safran évoque l'or, l'observatoire, le coucher
  • Veracity en serif italique = quelque chose de jugé, pas mesuré

02Pragmatisme mobile

  • Signaux LONG/SHORT/NEUTRAL en haut à droite de chaque carte (premier coup d'œil)
  • Bandeau global Régime / Liquidité / Stress en permanence
  • 3 drivers max par carte, sources nommées
  • Contre-scénario explicite, jamais caché
  • Bouton "Vue orbitale" en bas de Signals pour la métaphore visuelle complète
  • Sources avec sparklines et veracity affichée

03Hiérarchie de l'attention

  • Signal directionnel = élément le plus contrasté
  • Veracity = en serif italique pour souligner que c'est un jugement
  • Sources = en mono pour le caractère technique/factuel
  • État global = en italique poétique (sentiment du marché)
  • Étoiles = présentes mais discrètes, jamais distractives

04Choix techniques pour Expo

  • Le fond étoilé peut être une image SVG statique ou des dots CSS (perf OK)
  • Les orbes lumineuses = radial-gradient + box-shadow (natif RN)
  • La vue orbitale peut utiliser react-native-svg pour les positions
  • Fraunces + JetBrains Mono = à charger via expo-font
  • Compatible mode sombre uniquement (par design)
